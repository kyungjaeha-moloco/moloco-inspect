### [LINKEDIN]
Docker 컨테이너 안에서 `npm install`이 SSL 오류로 실패했습니다.

`CERT_UNTRUSTED`. 인증서를 신뢰할 수 없다는 오류인데, 로컬 머신에서는 아무 문제 없었어요. 컨테이너 안에서만 실패했습니다.

원인은 Zscaler였습니다. 회사 네트워크의 Zscaler 프록시가 HTTPS 트래픽을 중간에서 복호화하고 자체 인증서로 재서명합니다. 로컬 머신에는 이 CA 인증서가 설치되어 있지만, Docker 컨테이너는 신선한 Linux 환경이라 모르는 인증서예요.

해결책은 Zscaler CA 인증서를 Dockerfile에 주입하는 것이었습니다. `COPY zscaler-ca.crt /usr/local/share/ca-certificates/` 한 줄과 `update-ca-certificates`.

이틀 동안 VPN 설정, npm 설정, Docker 네트워크를 뒤졌는데, 답은 CA 인증서 한 줄이었어요.

기업 환경에서 Docker를 쓰신다면 Zscaler나 유사 프록시와의 충돌을 먼저 확인해보시길 권합니다.
