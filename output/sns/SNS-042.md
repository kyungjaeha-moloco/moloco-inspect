### [TWITTER]
AI가 만든 코드가 검증에 실패했다.

그러면 사람이 피드백을 줘야 한다.

auto-fix-loop — 검증 실패 시 AI가 스스로 원인을 분석하고 재시도한다. 사람이 중간에 끼지 않아도 된다.

#AI코딩 #자동화

---

### [LINKEDIN]
AI가 코드를 만들고 → 검증이 실패하면 → 사람이 피드백을 주고 → AI가 수정하는 루프.

이 사이클에서 사람이 병목입니다.

`auto-fix-loop`는 검증 실패 시 AI가 스스로 원인을 분석하고 재시도하는 자동화 루프입니다.

1. AI가 컴포넌트 구현
2. validate.ts 자동 실행
3. 실패 시: 에러 로그를 AI에게 다시 전달
4. AI가 원인 분석 후 수정
5. 재검증
6. 통과 시 완료 / 3회 실패 시 사람에게 에스컬레이션

이 루프를 붙이고 나서 제가 개입해야 하는 사이클이 약 60% 줄었습니다. AI가 혼자서 처리할 수 있는 수정들이 생각보다 많습니다.

사람의 역할은 루프를 설계하는 것, 루프 안의 모든 반복을 처리하는 것이 아닙니다.

반복 수정 작업, 어떻게 줄이고 계신가요?
