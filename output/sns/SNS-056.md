### [LINKEDIN]
디자인 시스템 사이트가 느렸습니다.

탭을 클릭할 때마다 관련 없는 컴포넌트까지 리렌더됐어요. React DevTools Profiler를 열어보니 원인이 명확했습니다. 메모이제이션이 없었어요.

두 가지를 적용했습니다.

첫째, `React.memo`. 자주 렌더되지만 props가 바뀌지 않는 컴포넌트에 감쌌습니다. 불필요한 리렌더가 즉시 사라졌어요.

둘째, `React.lazy + Suspense`. 탭 컴포넌트를 lazy import로 전환했습니다. 초기 번들 사이즈가 절반 가까이 줄었고, 첫 로딩이 눈에 띄게 빨라졌어요.

성능 최적화는 거창한 아키텍처 변경이 아닐 때가 많습니다. 올바른 위치에 두 줄 추가하는 것만으로도 체감이 달라집니다. 문제는 어디에 넣어야 하는지 아는 것이고, 그건 측정에서 시작합니다.

React 프로젝트에서 성능 이슈를 어떻게 찾으시나요?
