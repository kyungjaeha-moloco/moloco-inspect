### [LINKEDIN]
Docker 이미지에 npm 인증 토큰을 넣는 방법이 잘못되어 있었습니다.

처음엔 `.npmrc`를 그냥 COPY했습니다. 작동은 했습니다. 하지만 토큰이 이미지 레이어에 영구적으로 남습니다. `docker history`로 꺼낼 수 있습니다.

`docker build --secret` 플래그로 바꿨습니다. 빌드 중에만 시크릿을 마운트하고, 완성된 이미지에는 아무 흔적도 없습니다.

```
RUN --mount=type=secret,id=npmrc,target=/root/.npmrc \
    npm install
```

디자이너가 왜 이걸 알아야 하냐고요? AI 코딩 에이전트를 직접 운영하면서 Docker도 직접 다뤄야 했습니다. 보안은 개발자만의 일이 아닙니다.

사내 패키지 인증, 어떻게 처리하고 계신가요?
