### [LINKEDIN]
AI 에이전트 Sandbox를 재시작할 때마다 언어 설정이 영어로 돌아왔습니다.

한국어로 응답받으려면 매번 "한국어로 답해줘"라고 다시 말해야 했어요. 상태가 유지되지 않는다는 게 작은 것 같지만, 매 세션마다 컨텍스트를 다시 세팅해야 한다는 건 생각보다 비효율적이었습니다.

원인은 Sandbox 컨테이너가 시스템 로케일 없이 실행되고 있었기 때문이었어요.

Dockerfile에 환경변수를 추가했습니다.

```
ENV LANG=ko_KR.UTF-8
ENV LANGUAGE=ko_KR:ko
ENV LC_ALL=ko_KR.UTF-8
```

이후로 Sandbox를 재시작해도 언어 설정이 유지됩니다. AI도 처음부터 한국어로 응답해요.

개발 환경 설정은 한 번 맞춰두면 잊어버리기 쉽습니다. 그래서 Dockerfile에 명시적으로 넣는 것이 중요해요. 컨테이너는 항상 처음부터 시작하니까요.

개발 환경 언어/로케일 설정, 어떻게 관리하시나요?
