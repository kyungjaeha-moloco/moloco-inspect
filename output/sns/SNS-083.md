### [LINKEDIN]
AI가 작업 중일 때 사용자는 뭘 봐야 할까요?

"로딩 중..."으로 기다리게 하면 불안합니다. AI가 뭘 하는지 모르니까요.

Progress stepper를 만들었습니다. Setup → Working → Verify → Done. 각 단계에서 AI가 실제로 어떤 작업을 하는지 짧게 보여줍니다. "파일 분석 중", "코드 생성 중", "테스트 실행 중" 같은 식으로요.

사용자 테스트에서 차이가 명확했습니다. 단계가 보이면 기다리는 게 덜 답답합니다. 진행 상황을 알기 때문입니다.

AI UX의 핵심은 결과만이 아닙니다. 과정을 어떻게 보여주느냐입니다.

AI 작업 진행 상황을 사용자에게 어떻게 보여주고 계신가요?
