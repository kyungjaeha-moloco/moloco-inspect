### [LINKEDIN]
AI가 작업을 완료했다고 선언했을 때, 실제로 완료되어 있는 경우가 얼마나 될까요?

저의 초기 경험에서는 약 70% 정도였습니다. 나머지 30%는 레이아웃이 깨지거나, 반응형이 되지 않거나, 특정 상태에서 오류가 나거나, 다크 모드에서 색상이 맞지 않는 경우였습니다.

`preview-verification.json`은 AI가 작업 완료를 선언하기 전에 스스로 체크해야 할 항목들을 정의한 파일입니다.

- 모든 컴포넌트 상태에서 렌더링 확인
- 모바일/태블릿/데스크톱 3개 뷰포트 확인
- 다크 모드 확인
- 빈 상태/로딩 상태/에러 상태 확인
- 접근성 기본 항목 확인

이 체크리스트를 AI 컨텍스트에 넣자, "완성했습니다" 선언 전에 스스로 항목을 확인하고 미완성 부분을 먼저 보고합니다.

AI에게 "완성"의 기준을 알려주는 것. 이것도 설계입니다.
