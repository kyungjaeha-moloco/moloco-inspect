### [LINKEDIN]
코드 복사 기능을 구현한 코드를 열었습니다.

`document.execCommand('copy')`. Chrome DevTools에서 취소선이 그어져 있었어요. deprecated. 2015년에 표준에서 빠진 API입니다.

왜 살아 있었을까요. 동작했으니까요. 사용자 입장에서는 버튼 누르면 복사됐습니다. 문제가 없어 보이는 코드는 고치지 않게 됩니다.

`navigator.clipboard.writeText()`로 교체했습니다. 더 짧고, 비동기 처리가 자연스럽고, 브라우저 권한 모델과도 맞습니다.

동작하는 코드와 올바른 코드는 다릅니다. deprecated API는 언젠가 갑자기 죽습니다. 예고 없이, 특정 브라우저 버전 업데이트 이후에. 그때 가서 고치면 원인 추적이 힘들어요.

레거시 API, 지금 쓰고 계신 코드에도 숨어 있지 않나요?
